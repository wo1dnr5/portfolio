Tavily 검색 엔진으로 최신 AI 뉴스를 수집하고, Claude Sonnet 모델로 HTML 이메일 브리프를 자동 생성하여 발송하는 뉴스 자동화 파이프라인입니다. schedule 라이브러리로 주기적 실행을 설정하며, 수집·요약·발송 전 과정을 무인 자동화합니다. 이메일은 인라인 CSS가 적용된 모바일 친화적 HTML 형식으로 전송됩니다.

파이프라인 단계
  • Tavily Advanced 검색 (7일 이내 AI 뉴스 5건)
  • 기사 본문 600자 추출 및 구조화
  • Claude Sonnet HTML 이메일 생성 (4096 토큰)
  • 카드형 레이아웃 + 트렌드 총평 포함
  • Gmail SMTP 인라인 HTML 발송
  • schedule 라이브러리로 주기 자동화
이메일 구성 요소
  • 브리프 제목 + 날짜 헤더
  • 기사별 카드 (제목·2~3줄 요약·원문 링크)
  • 이번 주 AI 트렌드 총평 섹션
  • 인라인 CSS — 모든 이메일 클라이언트 호환
  • 최대 너비 600px 모바일 대응

Project 05
재무제표 분석 대시보드 — DART 공시 자동 분석
회사명만 입력하면 재무비율·성장성·분기 실적을 자동 계산해 인터랙티브 차트로 보여주는 Streamlit 웹앱

DART(금융감독원 전자공시)에서 상장기업 재무제표를 수집해 안정성·수익성·활동성·성장성 지표와 분기별 실적을 자동 산출하는 재무 분석 대시보드입니다. 데이터 수집 → 분석 엔진 → 시각화의 3계층 구조로 설계했으며, 한 번 받은 데이터는 parquet으로 캐싱해 재조회 시 API 호출 없이 즉시 로딩됩니다. Streamlit Community Cloud로 배포해 웹에서 바로 사용할 수 있습니다.

주요 기능
  • 회사명·종목코드 검색 (연결/별도 전환)
  • 분기 추세 — 매출·영업이익·순이익 + 이익률
  • 연간 분석 — 손익 추세·비율·전년 대비 성장률
  • 안정성 (부채비율·유동비율·자기자본비율)
  • 수익성 (영업이익률·순이익률·ROE·ROA)

  • 수집·분석·시각화 3계층 분리 아키텍처
  • parquet 로컬 캐시 — 재조회 시 API 호출 제거
  • IFRS 표준 계정 ID 우선 + 키워드 보완 추출
  • 분기 실적: Q4 = 연간 − (Q1+Q2+Q3) 산출
  • 삼성전자 연매출 333.6조원과 합계 일치 검증
